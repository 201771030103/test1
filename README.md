#####代码规范说明  
1.程序块要采用缩进风格编写，缩进的空格数为 4个。  
2.缩进或者对齐只能使用空格键，不可使用TAB键。  
3.相对独立的程序块之间、变量说明之后必须加空行。  
4.较长的语句（>80 ）要分成多行书写。  
5.不允许把多个短语句写在一行中，即一行只写一条语句。  
6.if 、for、do、while、case、switch 、default 等语句自占一行，且if、 for、do、while 等语句的执行语句部分无论多少都要加括号 {}。  
7.代码行之内应该留有适当的空格。
